# BrandKit Beta Acceptance Rubric

> Status: ready for a non-author trial. This file is not evidence that a trial occurred.

## Tester Protocol

1. Use a clean clone and only the README.
2. Do not receive live guidance from the repository author.
3. Build one Aether campaign and one Acme campaign offline.
4. Record every question, failure, manual fix, elapsed minute, and localized edit.
5. Evaluate at least two visual and two content artifacts from Acme.
